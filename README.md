# Jenkins

A Jenkins in Docker integrate with gosu and tini

## Feature
* gosu
* tini
* procps
* rsync
* detect uid of `/var/jenkins_home` (if mounted) and run as it

### Docker Pull Command

    docker pull albtdev/jenkins
